    <static> object()

    Creates a JavaScript object using the given arguments as a property list to initialize the object. There must be an even number of arguments -- one value for every key.

    Returns:
    The new object.
    An empty object
    >> (object)
    => {}
    An object using keywords as keys
    >> (object :start 0 :end 10)
    => {start: 0, end: 10}
    An object using strings as keys
    >> (object "start" 0 "end" 10)
    => {start: 0, end: 10}
    A compound object
    >> (object :name "Joe"
               :parents (array (object :name "John")
                               (object :name "Jane")))
    => {name: "Joe",
        parents: [{name: "John"},
                  {name: "Jane"}]}
