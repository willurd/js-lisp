<static> lisp.functions.join()
Joins the given arguments together into one string, using the first argument as the separator. 
Join items from a single list
  >> (join ", " (list "one" "two"))
  => "one, two"
Join items from multiple lists
  >> (let ((l1 (list "one" "two"))
           (l2 (list "three")))
       (join ", " l1 l2))
  => "one, two, three"
Returns:
The string result of the joined arguments.
